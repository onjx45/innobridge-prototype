# InnoBridge

InnoBridge is a prototype platform that connects university students with real industry problems, helping them gain hands-on experience, build portfolios, and collaborate on meaningful projects.

## Features
- User authentication
- Profile management
- Project posting
- Database integration
- Input validation for duplicate usernames and emails

## Tech Stack
- VB.NET (Windows Forms)
- Microsoft Access

## Purpose
To bridge the gap between academic learning and real-world industry collaboration.
